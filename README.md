# Windows Server Monitoring Scripts

This repository contains PowerShell scripts used to automate routine system administration tasks in a Windows Server environment.

## Features
- Disk space monitoring with configurable threshold
- Lightweight and easy to deploy
- Can be scheduled using Windows Task Scheduler
- Supports proactive system monitoring

## Script: disk-monitor.ps1
Checks disk usage and alerts when usage exceeds defined threshold (default: 80%).

## Use Case
Designed to reduce manual monitoring effort and improve system reliability in enterprise environments.
